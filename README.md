# 🤖 Machine Learning Algorithms Repository

A collection of Machine Learning algorithms implemented using Python and Jupyter Notebook. This repository demonstrates fundamental supervised learning techniques with practical examples and datasets.

## 📌 Algorithms Included

### 1. Linear Regression
- Predicts continuous values using a linear relationship between variables.
- Implemented using Scikit-learn.
- Notebook: `ML linear_regression.ipynb`

### 2. Simple Linear Regression (SLR)
- Basic regression model using a single independent variable.
- Notebook: `SLR More Accurate Data.ipynb`

### 3. Logistic Regression
- Classification algorithm used for binary outcomes.
- Notebook: `ML (logistic_reg).ipynb`

### 4. Polynomial Regression
- Extends linear regression by fitting polynomial relationships.
- Notebook: `polynomial Regression.ipynb`

### 5. K-Nearest Neighbors (KNN)
- Instance-based learning algorithm for classification.
- Notebook: `KNN.ipynb`

### 6. Decision Tree
- Tree-based classification model.
- Weather dataset example included.
- Notebook: `weather decision tree.ipynb`

### 7. Machine Learning Basics
- Introduction to Machine Learning concepts and workflow.
- Notebook: `Machine learning.ipynb`
